🌐 Portfolio Website

Ein modernes, responsives Web-Portfolio, entwickelt mit HTML, CSS und JavaScript, zur Präsentation meiner Projekte, technischen Fähigkeiten und meiner Entwicklung als Softwareentwickler.

👉 Ziel dieses Projekts ist es, eine klare, strukturierte und ansprechende Online-Präsenz zu schaffen, die sowohl technische Kompetenz als auch Designverständnis widerspiegelt.

📌 Projektübersicht

Dieses Portfolio dient als zentrale Plattform, um:

meine Projekte strukturiert darzustellen
meine technischen Fähigkeiten sichtbar zu machen
meine Entwicklung als Entwickler zu dokumentieren
eine Kontaktmöglichkeit für Recruiter und Unternehmen bereitzustellen

✨ Features
🧑‍💻 About Me – persönliche Vorstellung und Ziele
🛠️ Skills Section – Übersicht meiner technischen Fähigkeiten
📁 Projektübersicht – Darstellung ausgewählter Projekte
📬 Kontaktbereich – Möglichkeit zur direkten Kontaktaufnahme
📱 Responsive Design – optimiert für Desktop und mobile Geräte
🎨 Modernes UI – übersichtliches und benutzerfreundliches Layout
🧠 Technischer Ansatz

Die Anwendung basiert auf einer klassischen Frontend-Architektur:

🔹 HTML (Struktur)
Semantischer Aufbau der Website
Klare Trennung der Inhalte
🔹 CSS (Design & Layout)
Flexbox / ggf. Grid für Layout
Responsive Design mittels Media Queries
Fokus auf Lesbarkeit und Benutzerfreundlichkeit
🔹 JavaScript (Interaktivität)
Dynamische UI-Elemente
Event Handling (z. B. Navigation, Interaktionen)
Verbesserung der User Experience
